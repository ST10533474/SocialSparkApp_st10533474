# Social Spark App ST1533474

Description
The Social Spark App is an Android application that suggests simple social interaction ideas based on the time of day entered by the user.


Features
User enters a time of day (Morning, Afternoon, Dinner)
App displays a relevant social suggestion
Reset button clears input and output
Uses conditional logic (if/when statements)


How It Works
The app takes user input and uses conditional logic to determine which suggestion to display.

Morning → Send a "Good morning" text
Afternoon → Share a funny meme
Dinner → Call a friend
Night → Comment on a post


Technologies Used
Kotlin
Android Studio
XML (for layout design)


How to Run the App
1. Open the project in Android Studio
2. Run the app using an emulator or Android device
3. Enter a time of day
4. Click "Get Suggestion"
